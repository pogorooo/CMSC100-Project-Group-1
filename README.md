## How to Run

First, clone the repository in your desired directory
 
        git clone https://github.com/pogorooo/CMSC100-Project-Group-1.git
    


After cloning, run this to the terminal to get dependecies(you can use any of the two)

        npm i 
or

    npm install


Run the app using:
        
        npm start
